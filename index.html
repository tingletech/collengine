collengine by tingletech

collection engine: digital object description tool designed to run on google app engine

Simple Data Entry Engine for Digitization Projects. Designed to run on google app engine, but it does not suffer from "vendor lock in" because it is based on djangoappengine and django-nonrel.

Dependencies

python 2.5+ Google App Engine SDK

Install

clone the repo; edit the app.yaml to point at your google app engine account

License

modern 3-clause BSD

Authors

University of California

Contact

use issue tracker on github

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/tingletech/collengine

get the source code on GitHub : tingletech/collengine
